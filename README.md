# C++ Banana Screen Meltdown

## Overview

This project is an experimental realtime screen manipulation system developed in C++ using the Win32 API and GDI rendering techniques.

The system captures and distorts the user's desktop environment in realtime, creating procedural melting, displacement, and instability effects inspired by glitch aesthetics and surreal visual manipulation.

The “banana” theme introduces exaggerated color contamination and absurd visual layering to transform the desktop into an unstable audiovisual environment.

---

## Technologies Used

- C++
- Win32 API
- GDI Rendering
- Visual Studio

---

## Features

- Realtime screen capture
- Dynamic distortion effects
- Mouse-reactive visual deformation
- Procedural screen melting
- Experimental color contamination system
- Fullscreen rendering

---

## How to Run

1. Open the project in Visual Studio
2. Build the solution in x64 mode
3. Run the executable

---

## Project Goal

This work explores instability, digital decay, and absurdity through realtime visual computation.

The project is also part of a broader research interest in the relationship between human perception, interaction, and artistic experience.
